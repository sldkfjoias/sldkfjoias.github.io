计算机网络概述

发表于 2020-04-18 更新于 2021-04-08

概念、组成、功能、分类

计算机网络：一个将分散的、具有独立功能的计算机系统。通过通信设备和线路连接起来，由功能完善的软件实现资源共享和信息传递的功能。
计算机网络是互联、自治的集合。

功能：

  • 数据通信
  • 资源共享（软件、硬件、数据）
  • 分布式系统
  • 提高可靠性
  • 负载均衡

组成：

  • 硬件
  • 软件
  • 协议：规定了数据传输的规范

工作方式：

  • 边缘部分：用户直接使用
  • 核心部分：边缘部分服务

功能组成：

  • 通信子网：实现数据通信。是各种通信介质、通信设备、相应的网络协议组成。
  • 资源子网：实现资源共享/数据处理。是实现资源共享功能的设备和软件的集合

资源子网：应用层 表示层 会话层
通信子网：网络层 链路层 物理层

计算机网络分类

  1. 按分布范围分：

    • 广域网（交换技术）
    • 城域网
    • 局域网（广播技术）
    • 个人区域网

  2. 按使用者分：

    • 公用网络
    • 专用网络

  3. 按交换技术分：

    • 电路交换
    • 报文交换
    • 分组交换

  4. 按拓扑结构分：

    • 总线型
    • 星型
    • 环形
    • 网状型

  5. 按传输技术分：

    • 广播式网络：共享公共通信信道
    • 点对点网络：分组存储转发、路由选择机制

标准化工作

标准分类：

  • 法定标准：OSI
  • 事实标准：TCP/IP

制定标准流程：RFC（Request For Comments）

  • 因特网草案：此阶段还不是RFC文档
  • 建议标准：开始成为RFC文档
  • 草案标准：已取消该阶段
  • 因特网标准

国际标准化组织ISO： OSI参考模型、HDLC协议
国际电信联盟ITU：制定电气通信规则
IEEE：学术机构、IEEE802标准
IETF：RFC

速率、带宽、吞吐量

速率

速率：数据率或称数据传输率或比特率。
连接在计算机网络上的主机在数字信道上传送数据的位数的速率。
单位：b/s kb/s Mb/s Gb/s Tb/s $10^3$进制
1 Byte = 8 bit

带宽

带宽原指某个信号具有的频带宽度，即最高频率与最低频率之差，单位是赫兹（Hz）。
计算机网络中，带宽表示网络的通信线路传送数据的能力，通常是单位时间内从网络中某一点到另一点所能通过的最高数据率，单位是“比特每秒”
